# CalcVerse - All-in-One Calculator Hub

CalcVerse is a complete production-ready "All-in-One Calculator Hub" website built using Next.js 15 App Router, React, Tailwind CSS, TypeScript, and Framer Motion.

## Features
- **Modern UI/UX**: Dark mode + Light mode, Responsive, Smooth animations.
- **Calculators**: Finance, Education, Health, Technology, and Daily Utility.
- **SEO Optimized**: Dynamic sitemap, metadata, schemas, Core Web Vitals optimization.
- **Performance**: High Lighthouse scores, optimized images, fast loading.

## Tech Stack
- Frontend: Next.js 15, React, TypeScript, Tailwind CSS, Framer Motion.
- Libraries: next-seo, react-hook-form, zod, lucide-react, shadcn/ui.

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

3. **Build for production**:
   ```bash
   npm run build
   ```

## Folder Structure
- `src/app`: Next.js App Router routes and layouts.
- `src/components`: Reusable UI components.
- `src/lib`: Utility functions.
- `src/hooks`: Custom React hooks.
- `src/seo`: SEO utility functions and metadata generation.
- `src/types`: TypeScript type definitions.
- `public`: Static assets.

## Deployment
This project is optimized for deployment on Vercel. Ensure you have properly configured environment variables in your Vercel project settings before deploying.
